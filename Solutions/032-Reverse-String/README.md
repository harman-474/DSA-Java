# Reverse String

## Pattern

Two Pointers / Array Reversal

## Problem

Given a character array, reverse it in place.

## Approach

Use two pointers at the beginning and end of the array.

Swap the characters at both pointers, then move the pointers inward.

Continue until the pointers meet.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Key Learning

An array can be reversed in place by swapping elements from both ends using two pointers.
